Comandos para el uso de Flask

Creacion del entorno virtual
python -m venv env

Entrar al entorno virtual
.\env\Script\activate

Instalacion de las librerias de flask ya estando en el entorno virtual
pip install flask
pip install autopep8
pip install flask-script
pip install mysqlclient flask-mysql flask-mysqldb
pip install flask-login
pip install flask-bcrypt

Nota: Las librerias de flask se instalar dentro del entorno virtual, por ende debes de entrar primero al entorno virtual

Inicial proyecto
pip manage.py
